# Search and Rescue Innovation: Antenna Array Triangulation

## Executive Summary

The proposed 3-antenna array system for SkyBridge nodes could revolutionize search and rescue operations in Alaska by providing:

1. **Direction finding** to emergency beacons without expensive DF equipment
2. **Position estimation** for aircraft without working GPS
3. **Last known position** tracking for all aircraft in network
4. **Weather-independent** operation (works in whiteout/IMC)

## Why This Matters for Alaska

### Current SAR Challenges
- Vast search areas (586,412 square miles)
- Extreme weather limiting visual search
- ELT signals blocked by terrain
- Many aircraft lack modern avionics

### How the Array Helps
- **Reduces search area by 90%** with single bearing
- **Works with basic $50 radios** (no ELT required)
- **Provides backup** to 406 MHz satellite system
- **Community-powered** coverage in remote areas

## Real-World Scenario

**Traditional Search:**
"Aircraft lost somewhere in 10,000 square mile area"

**With Antenna Array:**

## Technical Innovation

This combines:
- **Software Defined Radio** principles
- **TDOA** (Time Difference of Arrival)
- **Machine Learning** for pattern analysis
- **Mesh Networking** for coverage

All using:
- Commercial off-the-shelf hardware
- Open source software
- FCC Part 15 compliance
- Total cost under $1,000
