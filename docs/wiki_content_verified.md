# SkyBridge Alaska - Wiki Content (Verified Facts Only)

## Project Overview

SkyBridge Alaska is a community-powered aviation safety mesh network designed to address critical infrastructure gaps in Alaska's aviation system. The project leverages NASA's TAIGA protocol for efficient data compression and uses LoRa mesh networking technology.

### Technical Specifications (Verified)

#### SkyBridge Technology
- **Frequency Band**: 902-928 MHz (FCC Part 15)
- **Technology**: LoRa mesh networking
- **Range**: Extended range capabilities at altitude
- **Power**: Solar-powered systems
- **Cost**: $50 per mesh node
- **Installation**: $200 per site (estimated)
- **Annual Maintenance**: $100 per site (estimated)
