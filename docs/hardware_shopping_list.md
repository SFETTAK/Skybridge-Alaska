# SkyBridge Alaska Hardware Shopping List

## Fixed Node Components (2 nodes total)

### Per Node:
- [ ] **Raspberry Pi**
  - Pi 4B (4GB) or Pi 5
  - MicroSD card (32GB+ industrial grade)
  - Official power supply or PoE HAT
  - Heat sinks/cooling

- [ ] **LoRa Radio for Node**
  - RAK WisBlock (recommended for nodes)
  - Or Heltec V3 with good antenna connector
  - USB cable for Pi connection

- [ ] **Antenna System**
  - 6-8 dBi outdoor omnidirectional antenna (915MHz)
  - Low-loss coax cable (LMR-400 or similar)
  - Lightning arrestor
  - Antenna mounting hardware

- [ ] **Enclosure**
  - IP65/67 weatherproof box
  - Cable glands for waterproofing
  - Mounting brackets

## Pilot Handheld Radios (10-20 units)

### Recommended Mix:
- [ ] **5-10x T-Echo** ($35-45 each)
  - Budget friendly
  - Great battery life
  - Good for basic users

- [ ] **5-8x T-Beam Supreme** ($65-85 each)
  - Built-in GPS
  - Proven reliability
  - Good for active pilots

- [ ] **2-3x Heltec V3** ($35-50 each)
  - Excellent performance
  - Nice display
  - Good for power users

### Accessories:
- [ ] USB-C cables (1 per radio)
- [ ] Protective cases (if available)
- [ ] External antennas for testing
- [ ] Battery banks for field use

## MQTT Server Hardware (if local)
- [ ] Small form factor PC or NUC
- [ ] UPS for power backup
- [ ] Ethernet connection to DOT network

## Tools & Supplies
- [ ] Coax crimping tools (if making custom cables)
- [ ] Multimeter for testing
- [ ] Label maker for device IDs
- [ ] Zip ties, velcro, mounting tape
