Network Components:

  • Aircraft Nodes: RAK4631 LoRa radios with GPS, weather sensors, and aircraft power integration ($50-100 per node)
  • Ground Stations: Raspberry Pi + LoRa with internet gateway, weather stations, and VHF SDR transcription ($425-545 per station)
  • LoRa Mesh: 902-928 MHz ISM band, 50+ mile range at altitude, store-and-forward capability
  • MQTT Broker: Central telemetry collection, node monitoring, and data distribution hub
  • Weather Integration: NOAA, NWS, FAA NOTAM feeds encoded with NASA TAIGA ASN.1 protocol
  • Mobile Apps: iOS/Android React Native apps with offline-first architecture
